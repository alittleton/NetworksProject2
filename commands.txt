Implement GET, PUT, and QUIT
	get:remote to local
	put:local to remote
	quit: quit

syntax: command:<args> e.g. GET:file.txt

PUT: client send STOR with filename
     server respond with CTS if no error
	filename exists, disk full, etc
     CONT message

GET: client send RTRV with filename
     server respond with RTS if no error
	no such file, wrong permissions, etc.
     CONT message

errors: ERR:XXX <error string>
	XXX is short error code, followed by a space

	display error description on stdout
